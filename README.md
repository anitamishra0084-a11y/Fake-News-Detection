Fake News Detection System
Project Overview
The Fake News Detection System is a machine learning-based project that classifies news articles as real or fake. It uses natural language processing techniques and a trained machine learning model to analyze text and make predictions.
Features
Classifies news as real or fake
Uses NLP techniques for text preprocessing
Machine learning model for prediction
Simple web interface for user input
Tech Stack
Python
Scikit-learn
Pandas
NumPy
Flask / Streamlit
HTML, CSS
